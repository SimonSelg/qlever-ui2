Backend Service

Wikidata Query Execution

Executes the Wikidata Query on the QLever Engine.

API

/api/qlever/<query>/<amount?>, where <query> is an base64url -encoded sparql query and the optional <amount?> controls the maximum amount of results to return. Default value is 100.
